# Contribuindo

Obrigado por seu interesse em contribuir para o projeto! Aqui estão algumas diretrizes:

## Como Contribuir

1. Faça um fork do repositório
2. Crie uma branch para sua feature (`git checkout -b feature/AmazingFeature`)
3. Commit suas mudanças (`git commit -m 'Add some AmazingFeature'`)
4. Push para a branch (`git push origin feature/AmazingFeature`)
5. Abra um Pull Request

## Padrões de Código

- Use ES6+
- Siga as convenções de nomenclatura (camelCase para variáveis, PascalCase para classes)
- Adicione comentários para código complexo
- Teste seu código antes de fazer commit

## Relatórios de Bug

Ao reportar um bug, inclua:

- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado
- Seu ambiente (Node version, OS, etc)

## Sugestões de Feature

Para sugerir uma nova feature, abra uma issue com:

- Descrição clara da feature
- Caso de uso
- Possível implementação (opcional)

## Dúvidas?

Abra uma discussion ou issue no repositório.
